
# Project Title


A modern AI-powered Developer Productivity Tracker that helps developers manage projects, track tasks, monitor GitHub activity, and gain AI-driven productivity insights through an elegant dashboard.

## 📌 Features
### 📊 Dashboard
- Real-time productivity overview
- Interactive charts and analytics
- Project statistics
- Task summary
- Recent project activity
- Productivity score
### ✅ Task Management
- Create tasks
- Update task status
- Delete tasks
- Task priorities
- Progress monitoring

### 🤖 AI Insights
- AI-powered productivity analysis
- Personalized recommendations
- Development insights
- Performance tracking

### 🐙 GitHub Integration
- Connect GitHub account
- Repository overview
- Contribution insights
- Developer statistics

### 👤 User Management
- Secure authentication
- JWT Authorization
- User profile
- Productivity tracking
- Current streak monitoring


## Tech Stack

**Frontend:** 
- React.js
- React Router DOM
- Axios
- Bootstrap
- React Toastify
- Chart.js
- React ChartJS-2

**Backend:** 
- JavaScript


## Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/devtrack-ai.git
## Backend Setup

terminal
cd backend
```


Run

```bash
npm run dev
```

Backend shows:
MongoDb Connected



---

## Frontend Setup

```bash
cd frontend
```



Run

```bash
npm start
```

Frontend starts on

```
http://localhost:3000
```
